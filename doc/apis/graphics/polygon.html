多边形 polygon

Canvas2D 

var polygon = $$.canvas.polygon('canvas')
    // 设置进度（即两点间的曲线段分成的段数，可选）
    .setNum(num);

获取多边形对象以后，直接传递点集合既可绘制多边形： 

polygon([[150, 100], [100, 250], [100, 500], [150, 500], [180, 400], [240, 400]]).fill();

SVG 

var polygon = $$.svg.polygon()
    // 设置进度（即两点间的曲线段分成的段数，可选）
    .setNum(num);

获取多边形对象以后，直接传递点集合既可绘制多边形： 

$$('path').attr('d',polygon([[150, 100], [100, 250], [100, 500], [150, 500], [180, 400], [240, 400]]).fill());
