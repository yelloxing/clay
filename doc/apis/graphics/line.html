曲线 line

可以绘制的曲线都有一个前置条件：是一个合法的一元函数，也就是说，一个x对应唯一的y值，且x值是递增的。

Canvas2D 

var line = $$.canvas.line('canvas')
    // 设置精度
    .setPrecision(dis)
    // 设置张弛系数
    .setU(u);

上面二个方法都不是必须的，你可以选择使用默认值。 

line(points).stroke();

points是一个数组，长度不小于2，里面的成员应该是一个数组，下面看看几个例子： 

line([1, 2])、line([[1, 2]])
line([[1, 2], [3, 4]])、line([[1, 2], [3, 4], [5, 6]])

SVG 

var line = $$.svg.line()
    // 设置精度
    .setPrecision(dis)
    // 设置张弛系数
    .setU(u);

参数说明依旧和上面是一样的，包括下面的points。 

$$('path').attr('d', line(points))
    .css({
        'key':'value',
        ...
});
